Git Bash 基本操作 windows用户可以下载一个 Git Bash,相当于windows里的命令提示符，这个软件可以直接对电脑里的文件进行操作，可以通过这个软件远程操作其他电脑。LinuX用户直接打开终端就行。 显示当前文件路径 pwd 查看当前文件内容（不包含隐藏文件） ls 查看当前文件内容（包含隐藏文件） ls -a 查看当前目录下所有文件详细信息（包括隐藏文件） ls -al 打开“music”文件 cd music 返回上一级文件目录 cd .. 删除/添加“音乐”文件夹 mkdir 音乐 rm -r 音乐 rm -r 音乐 强制删除，不提示 删除/添加“浏览.html”文件 touch 浏览.html rm 浏览.html 重命名文件/文件夹 mv a abc 即把a文件夹重命名为abc 文件的话操作相同但要加上后缀名 ~表示家目录 /表示根目录 关于VIM的使用 -打开vim vim a.md(没有a.md这个文件会自动创建，也可以打开别的文件) i 进入编辑模式 esc 退出编辑模式，进入命令模式 :qw 保存并退出 : q! 不保存退出
